# 将文本复制到剪贴板的快速指南

当用户需要将内容复制到剪贴板时，**直接执行 shell 命令**完成操作。

## 执行方式 (macOS)

```bash
echo -n '要复制的内容' | pbcopy
```

多行内容：
```bash
pbcopy <<'EOF'
第一行
第二行
EOF
```

## 规则

1. **必须执行命令**：不要只输出内容，要运行 `pbcopy` 将内容写入剪贴板
2. 执行成功后简短确认（如"已复制"）
3. 内容中有单引号时用 `$'...'` 或 heredoc 处理
4. 敏感信息先确认再执行
5. 不擅自修改用户提供的原文
